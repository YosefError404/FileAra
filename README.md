# 📂✨ FileAra 🚀

🖥️⚡ A modern, smart, and powerful file organizer built with 🐍 Python & 🎨 PySide6

FileAra helps you automatically 🗂️ organize, manage ⚙️, and clean ✨ your files and folders with a fast ⚡, beautiful 🌈, and user-friendly 🖱️ desktop experience.  
Designed for users who want a simple ✅ yet powerful 💪 way to keep their workspace neat 📚, organized 🗃️, and easy to access 🔍.

---

# ✨🔥 Features

🗂️ Automatic file sorting by file type  
📁 Smart category-based organization  
🔎 Recursive search in folders & subfolders  
🌙☀️ Beautiful Dark & Light themes  
🌍🇮🇷🇺🇸 Persian & English language support  
⚡ Fast, lightweight, and smooth performance  
🔄 Built-in update checker  
🧹 Empty-folder cleanup support  
🖥️ Modern and clean desktop interface  
⚙️ Customizable categories & file extensions  
📊 Simple and user-friendly dashboard  
💾 Easy management for large files  
🔒 Safe and secure file handling  
🚀 Optimized for productivity and speed  
🎯 Minimal, clean, and elegant design
